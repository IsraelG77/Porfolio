# People Analytics & Systems Portfolio
Interactive Data Analytics and HR Tech Portfolio demonstrating applied business logic, dashboard prototyping, and data visualization.

## 📌 Project Overview
This repository contains the source code for my professional portfolio and interactive dashboard demos. It is built as a lightweight, zero-dependency suite of standalone HTML files to enable fast viewing directly in the browser via GitHub Pages.

### Highlights:
* **Interactive NextGen Dashboard Demo:** A fully client-side prototype of a recruitment analytics dashboard.
* **100% Mock Data Generation:** Uses a custom JavaScript deterministic engine to generate realistic candidate pipelines, avoiding the need for a backend or exposing any PII.
* **Bilingual Support (PT/EN):** The entire UI and data logic logic implements internationalization to cater to global audiences.
* **Tech Stack:** Vanilla JavaScript, HTML5, CSS3, Chart.js, DataTables, and jQuery.

## 🏗️ Architecture Note
To ensure this demo is universally accessible without deployment pipelines (Node.js, React, etc.), the architecture intentionally uses a **Single-File Monolithic approach** (HTML, CSS, and JS combined). This demonstrates rapid prototyping capabilities and direct DOM manipulation, ideal for proofs-of-concept in People Analytics.
